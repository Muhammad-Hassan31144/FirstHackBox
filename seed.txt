# Project Brief: Red Team Command Reference Tool

## ğŸ¯ Project Overview
Create a comprehensive, searchable command reference tool for penetration testing and red team operations. The application should be a single-page React app with Tailwind CSS, featuring categorized commands for Windows and Linux systems with advanced search and filtering capabilities.

---

## ğŸ“ Technical Stack
- **Framework**: React (with JSX)
- **Styling**: Tailwind CSS (utility-first, responsive design)
- **Data Storage**: JavaScript objects/arrays (no backend needed)
- **Features**: Search, filter, copy-to-clipboard, syntax highlighting

### Design Specifications

**Color Scheme** (Dark theme optimized for security tools):
- Background: `bg-gray-900`
- Cards: `bg-gray-800`
- Accent: `bg-blue-600` (for buttons/highlights)
- Text: `text-gray-100`
- Muted text: `text-gray-400`
- Code blocks: `bg-gray-950` with `text-green-400`

**Typography**:
- Headers: `font-bold text-xl`
- Command syntax: `font-mono text-sm`
- Body: `text-base`

**Spacing**:
- Card padding: `p-4`
- Card gaps: `gap-4`
- Section margins: `mb-6`
